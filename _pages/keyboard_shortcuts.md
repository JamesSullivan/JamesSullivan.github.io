#### Jupyter (in VS Code)

- `esc` - command mode j/k down/up  m/y markdown/code
- `enter` - edit mode in cell from command mode
- `Ctrl+Enter` - to run the current cell
- `⇧+Enter` - run the current cell and advance to the next.
